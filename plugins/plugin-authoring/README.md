# plugin-authoring

Tools for building Claude Code extensions — agents, skills, and plugins.

## Agents

| Agent | Role |
|-------|------|
| agent-creator | Creates agent configurations from user requirements |
| plugin-builder | Scaffolds complete new plugins with directory structure, manifest, agents, skills, and README |
| plugin-validator | Validates plugin structure and manifests |
| hook-author | Creates hooks (hooks.json config + scripts) for plugins and projects |
| conversation-analyst | Analyzes conversation to identify skill, agent, and hook opportunities |
| skill-manager | Manages existing skills — inventory, validate, organize, improve, package |

## Skills

| Skill | Purpose |
|-------|---------|
| agent-skills | Create, evaluate, and improve agent skills |
| create-agents | Guide for creating effective subagents |
| write-for-llms | Write precise prose optimized for LLM consumption |

## Install

```
/plugin marketplace add plugin-authoring
```
